Proyectos

Pokédex web

El proyecto consiste en el desarrollo de una Pokédex web que utiliza la PokeAPI, una API pública que proporciona información sobre los Pokémon. La aplicación permite al usuario buscar un Pokémon por su nombre o número, realiza una solicitud a la API mediante JavaScript y fetch, y procesa la respuesta en formato JSON para mostrar datos

API de Deezer

El proyecto consiste en el desarrollo de una aplicación web que consume la API de Deezer, una plataforma de música en línea. La aplicación permite buscar canciones, artistas o géneros musicales y mostrar información como el título de la canción, el artista, la portada del álbum y una vista previa de audio.

API desarrollada en PHP

La práctica consiste en modificar una API desarrollada en PHP para que la respuesta en formato JSON incluya el nombre del alumno enviado mediante un parámetro en la URL. Para lograrlo, se utiliza la variable superglobal $_GET, que permite recibir datos desde el navegador a través del query string. De esta forma, el programa toma el nombre enviado en la dirección web y genera un mensaje dinámico personalizado.

Herraminetas utilizadas

PHP

lenguaje de programación utilizado para crear y modificar la API y generar la respuesta en formato JSON.

(como Google Chrome o Edge): para enviar los parámetros en la URL y visualizar la respuesta de la API.

Editor de código

(por ejemplo Visual Studio Code o Notepad++): para escribir y modificar el código PHP.
